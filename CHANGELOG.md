## [1.0.0] - 2025-01-11

### Added

- Core data management system with automatic scanning and registration
- `[ManagedData]` attribute for marking ScriptableObject types
- Visual editor window (Tools/SO Data Manager, Ctrl+Shift+M)
- Real-time search and filtering by name, type, and category
- Advanced multi-condition search functionality
- Batch operations with multi-select mode
- Batch edit multiple assets at once
- Export all asset paths to text file
- Reference finder - locate which objects reference selected asset
- Dependency viewer - visualize asset dependency graph
- Orphan detection - find unused assets
- Quick edit window with full Inspector integration
- SOHelper module with history navigation
- Quick open button (üîç) for all ScriptableObject reference fields
- Runtime and Editor assembly definitions
- Chinese and English documentation
